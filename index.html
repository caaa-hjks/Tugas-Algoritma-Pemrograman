1. Nested Loop

Nested Loop adalah perulangan di dalam perulangan lainnya.

for i in range(3):
    for j in range(3):
        print("*", end=" ")
    print()
Output: 
* * *
* * *
* * *

2. Do While

Perulangan yang dijalankan minimal satu kali sebelum kondisi diperiksa.

angka = 1
while True:
    print(angka)
    angka += 1
    if angka > 5:
        break
Output: 
1
2
3
4
5

3. While

Perulangan yang berjalan selama kondisi bernilai benar.

angka = 1
while angka <= 5:
    print(angka)
    angka += 1
Output: 
1
2
3
4
5

4. Struktur For

Perulangan dengan jumlah iterasi tertentu.

for i in range(1,6):
    print(i)
Output: 
1
2
3
4
5

5. Array

Pengertian Array

Array adalah struktur data yang digunakan untuk menyimpan sekumpulan data yang memiliki tipe data yang sama dalam satu variabel. Setiap data memiliki indeks sehingga dapat diakses dengan mudah.

Jenis-Jenis Array

1. Array Satu Dimensi

Array yang hanya memiliki satu indeks.

nilai = [80, 85, 90, 75, 88]

2. Array Dua Dimensi

Array yang memiliki baris dan kolom seperti tabel.

matriks = [
 [1,2,3],
 [4,5,6]
]

3. Array Multidimensi

Array yang memiliki lebih dari dua dimensi.

Struktur Dasar Array

  • Nama Array
  • Tipe Data
  • Indeks
  • Elemen Data
angka = [10,20,30,40,50]

Contoh Program Array

nilai = [80,85,90,75,88]

for i in nilai:
    print(i)
Output: 
80
85
90
75
88
